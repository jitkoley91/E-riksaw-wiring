⚠ CRITICAL: CHARGER COMPATIBILITY
A standard tubular battery charger CANNOT be used directly on lithium batteries. When replacing Lithium → Tubular, ensure the new charger is set to tubular/wet cell profile. Do not use a lithium charger on a tubular battery — it lacks the equalization cycle needed and may undercharge.
Step-by-Step Replacement Procedure
⚠ BEFORE YOU START
Switch OFF inverter and all loads. Disconnect AC mains. Allow lithium battery to discharge to 20% before removal. Wear rubber gloves and safety glasses. Have baking soda solution ready for acid neutralization.
01
Disconnect & Isolate Power
Turn OFF the main MCB. Disconnect the inverter from all AC loads. Switch off the charger. Wait 5 minutes for capacitors to discharge. Use a multimeter to verify zero voltage at the DC bus before touching any terminals.
CRITICAL
02
Remove Lithium Battery
Disconnect negative (−) terminal first, then positive (+). Label all wires before removing. Photograph the existing wiring. Remove the BMS unit and store separately — it will not be reused with tubular batteries. Note the cable sizes (typically 16–25mm² for 100Ah systems).
CAUTION
03
Prepare Tubular Battery Location
Tubular batteries must be placed in a ventilated enclosure or open area — never in a sealed cabinet. The location must allow hydrogen gas to escape freely. Ensure the floor can support the weight (a 150Ah tubular battery weighs ~40kg). Place on an acid-resistant tray or wooden pallet.
VENTILATION
04
Install Tubular Battery
Place tubular battery in prepared location. For 24V system: connect B1(−) to B2(+) with the series link cable (use 35mm² cable, ≤30cm length). Keep the series link as short as possible to minimize resistance. Ensure vent caps are tight but not overtorqued.
PROCEDURE
05
Connect DC Wiring
Connect positive cable (red, 25mm² minimum) from battery B1(+) → ANL fuse → inverter DC input (+). Connect negative cable (black, 25mm²) from battery B2(−) → inverter DC input (−). Use compression lugs — not push-in connectors. Torque terminal bolts to 8–10 Nm. Connect positive LAST.
POLARITY CHECK
06
Replace / Configure Charger
Install a dedicated tubular battery charger. Configure: Bulk = 14.8V (12V) / 29.6V (24V), Absorption = 14.4V, Float = 13.6V. Set temperature compensation if the charger supports it (−3mV/°C/cell). Connect charger AC input to MCB output. Connect charger DC output to battery terminals — positive to B1(+), negative to B2(−).
CONFIG
07
Install Earth / Ground Connections
Connect battery case earth to the main earth bus bar using green/yellow 6mm² cable. Connect inverter chassis earth to the same earth bus. Run a dedicated earth from the MCB earth terminal to the bus bar. All earth connections must be to a common point — no daisy-chaining.
EARTH
08
Install Temperature Sensor
Attach the NTC temperature sensor (usually supplied with charger) to the side of the battery body — mid-height, away from terminals. Route signal cable to the charger's TEMP input. This enables temperature-compensated charging which significantly extends tubular battery life.
OPTIONAL+
09
Verify & Power On
Double-check all polarity connections. Verify ANL fuse is seated. Check all terminal bolt torque. Turn ON MCB → charger should start and display bulk charging. Check charger output voltage with multimeter (should match set voltage). Check for any unusual heat or sparks. After 30 minutes, check battery temperature — should be below 40°C.
VERIFY
10
Initial Equalization Charge
New tubular batteries benefit from an equalization charge before first use. Set charger to equalization mode (15.5V / 31V for 12V/24V systems) for 6–8 hours. This ensures all cells within each battery are at the same state of charge. Do this monthly thereafter for best lifespan.
