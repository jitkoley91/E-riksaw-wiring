Complete System Wiring — E-Rickshaw 48V Li-NMC
⚠ CRITICAL — READ BEFORE STARTING 48V Li-NMC batteries are DANGEROUS without a BMS. Li-NMC cells can thermal runaway if overcharged (cutoff: 4.2V/cell). Never bypass the BMS. Use only a Li-NMC compatible 54.6V charger. Disconnect negative terminal first when removing old tubular batteries.

📋 SYSTEM ARCHITECTURE 13S4P Li-NMC pack: 13 cells in series = 48.1V nominal / 54.6V full charge / 36.4V cutoff. 4P = 4 cells in parallel per group = 120Ah total. BMS sits between pack and all loads — B+/B− connect to cells, P+/P− connect to controller and charger. DC-DC converter steps 48V down to 12V for lights, horn, indicators.
BMS Wiring — 48V 150A Li-NMC Protection
⚠ Li-NMC SPECIFIC DANGER Li-NMC cells are more energetically dense than LFP and have a narrower safe voltage window (2.8V–4.2V per cell). Overcharge beyond 4.25V/cell causes thermal runaway. The BMS must NEVER be bypassed or shorted. Use a BMS rated for at minimum 1.5× the controller's peak current.

Replacement Procedure — 4×12V Tubular → 48V Li-NMC
⚠ BEFORE YOU START Switch OFF the e-rickshaw key/ignition. Remove the key. Engage parking brake. Disconnect the motor controller first — before touching any battery terminals. 48V can cause fatal electric shock — never work alone. Tubular batteries contain dilute sulfuric acid — wear gloves and goggles.
01
Full System Isolation
Turn key OFF, remove key. Locate the main contactor or fuse link between battery and controller — disconnect it. Wait 5 minutes for controller capacitors to discharge. Verify 0V at controller DC input terminals using a multimeter. If rickshaw has an emergency cutoff switch, engage it.
CRITICAL
02
Remove Old Tubular Batteries
Disconnect 4 inter-battery series link cables first. Then disconnect the negative (−) main cable, then positive (+) cable. Remove one battery at a time — each 12V 120Ah tubular weighs ~30kg. Use proper lifting. Do not tilt batteries — acid spill risk. Dispose at certified lead-acid recycling centre. Clean battery tray area with baking soda solution to neutralise any acid residue.
ACID HAZARD
03
Prepare Battery Bay for Li-NMC Pack
Li-NMC pack is significantly lighter and smaller. Fabricate or source a mounting tray/bracket that secures the new pack firmly — the pack must NOT shift during vehicle movement. Ensure the pack enclosure is IP54 or better rated for outdoor use. No ventilation required (sealed cells), but avoid placing near direct exhaust heat. Line mounting area with anti-vibration foam padding.
MOUNTING
04
Install BMS (If External — Not Built-In)
Mount 48V 150A BMS in a protected location on the frame. Connect cell sense wires S0 to S13 (14 wires) to each cell group junction — follow the BMS manual pin order exactly — wrong pin order will damage BMS instantly. Connect NTC temperature sensors (×3) to cell body surface at positive, middle, and negative ends of the pack. Connect B+ and B− to pack terminals. Do NOT connect P+/P− yet.
BMS FIRST
05
Replace Charger — 54.6V Li-NMC Only
The old 48V lead-acid charger charges to 58V and has a bulk/absorb/equalise profile — this WILL overcharge Li-NMC cells and cause thermal runaway. Install a new 54.6V CC-CV Li-NMC charger (charge current 10–20A). Verify charger output voltage with multimeter before connecting to battery: must show 54.6V ± 0.2V at no-load. Never use equalisation mode — Li-NMC has no equalisation stage.
CHARGER CRITICAL
06
Reconfigure Motor Controller Settings
Access controller programmer (if available) or set via display: Low voltage cutoff = 39.0V (3.0V/cell × 13). Low voltage warning = 42V. High voltage limit = 54.6V. If controller has battery type selection, set to "Lithium" or "Li-Ion". If regen braking is enabled, ensure regen cutoff voltage = 54.0V (BMS will also cut regen if pack is full). Record all old settings before changing.
CONTROLLER
07
Wire DC Main Cables
Install ANL fuse (150A, 60V DC) within 30cm of BMS P+ terminal using a proper ANL fuse holder. Run red 35mm² flexible cable: BMS P+ → ANL fuse → main contactor → motor controller DC+. Run black 35mm² cable: BMS P− → negative bus → motor controller DC−. Also tap off from bus for DC-DC 48V→12V converter for lights and horn. Use tinned copper compression lugs — torque all bolts to 10 Nm. Connect positive LAST.
WIRING
08
Chassis Earth Bonding
Connect BMS enclosure, battery pack enclosure, motor controller case, and charger case to the rickshaw steel frame using 6mm² green/yellow cable. All earth connections to a single frame earth point (star connection) — not daisy-chained. This protects against insulation faults creating shock hazard on the vehicle body. Check frame-to-earth continuity <1 ohm with multimeter.
EARTH
09
First Power-On & Verification
With charger DISCONNECTED: turn key ON. Controller should power up. Check BMS display shows pack voltage (should be ~48–52V if pack is partially charged). Check all cell voltages on BMS display — all should be within 50mV of each other. Slowly turn throttle — motor should respond smoothly. Check for any fault codes. Then connect charger — verify charging starts at 54.6V max. Monitor temperature for 30 minutes of first charge.
VERIFY
10
Field Test & Capacity Break-In
Li-NMC cells benefit from 3–5 full charge/discharge cycles to reach rated capacity. Do a 5–10 km test run monitoring: BMS fault LEDs (should be off), motor temperature, controller temperature, and pack voltage under load. Verify regen braking works (if equipped). After first full cycle, check cell balance on BMS — all cells should be within 20mV. If any cell drifts more than 50mV, contact the BMS balancing function.
COMMISSION
